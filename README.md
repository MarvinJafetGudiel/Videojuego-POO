# Videojuego-POO
Tarea desarrollado en C# para aplicar los principales conceptos de la Programación Orientada a Objetos (POO) mediante la simulación de un sistema de personajes de videojuego.

# Descripción

Este proyecto consiste en la creación de un sistema de personajes para un videojuego utilizando Conceptos basicos de la programacion orientada a onjetos.

Se implementa una clase base Personaje y clases derivadas como Arquero, permitiendo definir características y comportamientos distintos para cada tipo de personaje.

# Conceptos de POO Aplicados

En este proyecto se utilizan múltiples conceptos fundamentales de la Programación Orientada a Objetos:

1.Clases y Objetos

2.Encapsulamiento y propiedades

3.Herencia

4.Polimorfismo

5.Constructores
